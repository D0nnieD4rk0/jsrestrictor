FingerPrint Detector Report

This report lists all JavaScript API endpoints that were accessed by the web page and are considered critical in terms of browser fingerprinting. FingerPrint Detector (FPD) uses this data to decide whether the page is collecting a browser fingerprint and how severe the fingerprint might be. Individual endpoints are assigned to groups based on their semantic properties and show a number of page accesses in brackets.

Default behavior is set to show only groups and endpoints that were accessed at least once. If you want to show all groups and their respective endpoints used by FPD, click here.
